--------------------------------------------------------------
Sign up
--------------------------------------------------------------

Route:
/api/signup

Example request body:
{
	"Username": "Username",
	"Password": "12345",
	"FirstName": "First",
	"LastName": "Last",
	"Email": "email@test.com",
	"PhoneNumber": "111-111-1111"
}

Example response:
{
    "success": true,
    "message": "Sign up successful!"
}

--------------------------------------------------------------
Login
--------------------------------------------------------------

/api/login

Example request body:
{
	"Username": "Username",
	"Password": "12345",
}

{
    "success": true,
    "message": "Login successful!",
    "token": "xxx.yyy.zzz"
}
